# alfred 有道翻译 插件
* 文本翻译
* 图片翻译
* 开箱即用

目前还有很多没有完善。

[本插件开发博客](https://github.com/coppyC/blog/issues/32)

# 文本翻译
* 智能翻译：中英互译，支持其它语言翻译成中文
* 快捷键呼出：受 alfred 限制，需要额外设置
* `shift` + `enter` 转大驼峰，方便写代码
<!-- * 英文发音 -->

关键词 `yd`

# 图片翻译
* 通过截图翻译图片，识别后的文字和翻译保存在剪切板中，可打开剪切板历史查看
* 支持快捷键呼出：需要设置

关键词 `ocr`

截图尽量简练，接口有图片大小2M的限制。
